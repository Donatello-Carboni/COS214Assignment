\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classWaitingToSit}{Waiting\+To\+Sit}} class represents the state where a customer is waiting to be seated in a restaurant. 

This class inherits from the base class \textquotesingle{}\mbox{\hyperlink{classState}{State}}\textquotesingle{} and defines methods related to different actions a customer can perform in this state.
